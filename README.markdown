DistrictBuilder
===============

[DistrictBuilder](http://www.districtbuilder.org/) is software created by the [Public Mapping Project](http://www.publicmapping.org/resources/software).


Installation
------------

### Requirements

Vagrant 1.8.1
VirtualBox 4.3
Ansible 2.2

### tl;dr

```bash
$ ./scripts/setup
$ vagrant ssh
$ ./scripts/update
```
